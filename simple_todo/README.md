Django-Simple-Todo
==================

simple-todo dajngo版

使用的环境： fedora 17 + django1.10 + python2.7 + sqlite3

前端使用的 bootstrap + jquery

实现添加修改删除 ajax支持 可自行添加用户控制

## 使用方法

clone 下代码 只需执行下面的命令

    python manage.py makemigrations todo
    python manage.py migrate
    python manage.py createsuperuser
